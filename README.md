# Postman Library API Tests

Postman collection for API testing (positive + negative scenarios).

## What’s included
- Status code + response body validations
- Positive and negative test cases
- Basic data checks (IDs, required fields, error messages)

## How to use (Postman)
1. Import the collection JSON from the repo into Postman
2. Import an environment file and set variables
3. Run using Collection Runner

## Repo structure
- `collections/` — Postman collection JSON
- `environments/` — sample environment (no secrets)
